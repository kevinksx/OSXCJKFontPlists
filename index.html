OSXCJKFontPlists

该Git专案用于寄存一些现行修改好的OS X字体回退档案（包括原厂未修改的档案），
且全部转为XML明文格式（不妨碍使用）。
这些修改都是针对诸如思源黑体或者冬青黑体等第三方字体的相容性而进行的。

请且仅请在您了解这些档案的适用目的之后再考虑下载这些档案。

附注：用curl下载github的档案、并自动套用正确的系统档案权限的方法：

sudo curl -L 「GITHUB档案地址」\?raw\=true -o 「目标档案名称（包含路径）」

例如：

sudo curl -L https://github.com/ShikiSuen/OSXCJKFontPlists/blob/master/README.md\?raw\=true -o ~/Desktop/"Hello World.txt"

This Git Project is for hosting customized OS X CJK Font Plists.

Not only Factorial version (converted into XML format),
but also modified version to fit 3rd-party fonts like Hiragino Sans or SHS.

You should download files only if you know what they are and what they should be used for.

You could use Terminal command for batch download purposes (with correct System Permission settings):

sudo curl -L 「GITHUB FILE ADDRESS」\?raw\=true -o 「TARGET FILENAME WITH DIRECTORY」

eg:

sudo curl -L https://github.com/ShikiSuen/OSXCJKFontPlists/blob/master/README.md\?raw\=true -o ~/Desktop/"Hello World.txt"
